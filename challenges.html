CURRENT CHALLENGE: Using reef-safe sunscreen.

Summer is here, and that means that you should be wearing sunscreen more than ever! But is it destroying coral reefs?

What does reef-safe mean?

Reef-safe simply means that it does not contain certain chemicals that cause coral bleaching, while still blocking UV rays.

How can I make sure my sunscreen is reef-safe?

Many sunscreens explicitly state that they are reef-safe, but unfortunately, this term is not regulated, so it is important to check the ingredients, even if the sunscreen markets itself to be reef-safe. Make sure your sunscreen does not contain oxybenzone, octinoxate, octocrylene, homosalate, any parabens, or any other ingredient on the HEL list.
